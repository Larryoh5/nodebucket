What is NodeBucket?

Initially developed for the sole purpose of helping organize an incredibly forgetful programmer Larry Ohaka, Nodebucket has evolved into a user-base platform for managing tasks for a fast paced lifestyle. With Nodebucket users are able to input there tasks, reorder them in terms of importance, and delete them when no longer needed. Also they are capable of placing tasks into the 'Done' column, when the user has completed the task. Using the signin feature users can use Nodebucket to save there tasks into our database so no data is loss for future use. Our main mission is to give our users an easy and reliable way of managing day-to-day tasks in a practical way
